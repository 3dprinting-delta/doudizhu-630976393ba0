    Rules and Notes

    How this web build works

    The original Pygame project dealt 17 cards to each player, used bidding to assign the landlord, and let simple AI agents make conservative plays. This site keeps that structure and ports the combination checker and AI decision rules into JavaScript.

    • The player holding the heart 3 opens the bidding round.
    • Rocket beats everything. Bomb beats every non-rocket hand.
    • If two players pass after a valid play, the table resets and the next player may lead anything.
    • The browser version supports the same main hand types recognized by the upstream game.
